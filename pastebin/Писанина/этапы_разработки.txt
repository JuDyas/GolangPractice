Изучение материала (2ч)

Настройка окружения (2-3ч)
    Создание директорий.
    Инициализация модулей.
    Настройка подключения к MongoDB.
    Создание базового сервера.

Реализация базового функционала (4-8ч)
    Обработка POST запросов.
    Сохранение данных в бд.
    Генирация ссылок для доступа.

Добавление авторизации (5-7ч)
    Регистрация/Авторизация.
    Генерация токенов (JWT).
    Проверка токенов.

Реализация ролей (6-8ч)
    Админ/смертный юзер.
    Админ функционал (отображение всех ссылок, пагинация, фильтрация, сортировка, удаление)

Docker и Makefile (2-4ч)
    Написание Dockerfile.
    Создание docker-compose.yml для MongoDB.
    Настройка Makefile для удобного запуска.
