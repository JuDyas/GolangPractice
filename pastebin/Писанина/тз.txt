Http Сервер для обмена текстовой информацией (pastebin)
После отправки текста пользователем сервер сохраняет его и отправляет ссылку, по которой его можно будет открыть. :)

Ссылки:
    Открытая: Любой человек, у кого есть ссылка, может её открыть. :)
    Ссылка с TTL:  Работает только определённое время (от 1 секунды до бесконечности). :)
    Ссылка с паролем. :)
    Ссылка для конкретного IP: Только пользователи с указанного IP-адреса могут открыть текст.
    Ссылка для авторизированных юзеров: Нужно войти на сайт, чтобы получить доступ.
    Ссылка для конкретного пользователя: Доступ только для одного конкретного зарегистрированного пользователя (по его email).

Авторизация и регистрация:
    Пользователи могут регистрироваться с помощью email и пароля. После использовать их для авторизации. (JWT токен)
    Подтверждение email не нужно.
    Возможно интеграция с Google auth.

Типы юзеров:
    Стандарт: Может создавать ссылки для конкретных людей (например, для друга).
    Админ:
        Может удалять ссылки.
        Может видеть все записи с возможностью:
            Сортировать (например, по дате создания).
            Фильтровать записи.
            Просматривать данные с пагинацией.

Удаление записей: Записи не удаляются физически. Вместо этого добавляется пометка, что запись удалена (soft delete).
Обновлять записи нельзя: Если запись создана, она остаётся неизменной.

Упаковать в докер и залить на Docker hub.

Технологии:
- Docker, MongoDB, Makefile, Golang, REST API, HTTP, Auth, JWT, (Google Auth?), Soft Delete, Pagination, Sorting, Filtering, Hashing, IP
