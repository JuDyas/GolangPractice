Задача
Написать http сервер для парсинга данных с Rozetka
(https://rozetka.com.ua/ua/notebooks/c80004/) через указанные параметры.
Реализовать endpoint который будет скрапить данные нужно по 4-м параметрам
(от дешевых, дорогих, новинки, за рейтингом) на выбор, так же скрапить нужно n элементов,
и возможностью пропуска товаров. Обработку данных нужно реализовать через механизм
переодических запусков задач (cron), Всю информацию зашифровать (Виженер)
использовать стороннюю библиотеку. Обработанные данные нужно будет сложить в redis.

Доступ к данным
Добавить endpoint для получения списка данных + пагинация по ним (без деталей)
Добавить endpoint для получения данных с деталями (использовать ключ для
декодировки данных)

Информация которая нас интересует
Екран, Процесор, Оперативна пам'ять, Накопичувачі даних, Відеокарта.
